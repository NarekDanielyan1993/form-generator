Vue 3 Form Generator

A reusable form generator component built with Vue 3, TypeScript, Vuex, Vue Router, and SCSS, supporting dynamic fields, slots, and state persistence.

Installation

Clone the repository:

git clone git@github.com:NarekDanielyan1993/form-generator.git
cd vue-form-generator

Install dependencies:

npm install

Development

Run the development server:

npm run dev

Open your browser at http://localhost:5173

Production Build
npm run build
npm run preview
